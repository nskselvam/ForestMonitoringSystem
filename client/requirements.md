## Packages
react-leaflet | Map visualization
leaflet | Map core library
@types/leaflet | TypeScript definitions for Leaflet
three | 3D rendering core
@react-three/fiber | React renderer for Three.js
@react-three/drei | Useful helpers for React Three Fiber
recharts | Charts and graphs
lucide-react | Icons

## Notes
- `leaflet` CSS needs to be imported for the map to render correctly.
- 3D visualization is a simulated terrain to demonstrate the concept without requiring complex external 3D assets.
- Initial dashboard loads with mock data to show structure; running analysis via the form triggers the API.
